Parágrafos e quebras de linhas

Tudo que for escrito dentro de paragrafos sera mostrado em sequência, mesmo que seja escrito em linhas diferentes.

Para criar um paragrafo novo deve-se utilizar o comando <br>,
exatamente assim.

Simbolos especiais:
® © ™ € £ ¥ ¢ Δ δ
E muitos outros
